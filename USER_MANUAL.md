# 社区诊所慢病随访管理系统 - 用户操作手册

## 一、系统概述

本系统是专为社区诊所设计的慢性病随访管理系统，帮助医生高效管理高血压、糖尿病等慢性病患者，实现患者档案管理、随访计划安排、健康指标记录、异常提醒和数据可视化等功能。

## 二、登录系统

1. 打开浏览器访问系统地址（如 http://localhost:3000）
2. 在登录页面输入用户名和密码
3. 点击「登录」按钮进入系统

默认测试账号：
- 管理员：admin / doctor123
- 普通医生：doctor1 / doctor123

## 三、首页仪表盘

登录后进入首页仪表盘，包含以下内容：

1. **统计卡片**：
   - 患者总数：当前管理的所有患者数量
   - 今日随访：今日计划随访的患者数量
   - 待处理计划：所有待随访的计划数量
   - 异常提醒：未解决的异常指标提醒数量

2. **最近随访记录**：展示最近的5条随访记录

3. **待处理随访计划**：展示今日和明日的待随访计划

4. **最近异常提醒**：展示高严重程度的异常提醒

## 四、患者管理

### 4.1 患者列表

点击左侧菜单「患者列表」进入患者管理页面：

**筛选功能**：
- 按姓名搜索：在搜索框输入患者姓名进行模糊搜索
- 按疾病类型筛选：选择高血压、糖尿病等疾病类型
- 排序：按姓名、年龄、创建时间排序

**操作**：
- **新增患者**：点击「新增患者」按钮，在弹窗中填写患者信息
- **查看详情**：点击患者姓名或「查看」按钮进入患者详情页
- **编辑患者**：点击「编辑」按钮修改患者信息
- **删除患者**：点击「删除」按钮（需二次确认）

### 4.2 患者详情

进入患者详情页后，可通过标签页切换查看不同信息：

#### 基本档案
- 查看患者完整信息（姓名、性别、年龄、联系方式、疾病类型、过敏史、既往病史等）
- 点击「编辑」按钮可修改患者档案

#### 健康指标
- 查看历史健康记录表格（血压、血糖、心率、胆固醇、甘油三酯、体重等）
- 点击「新增记录」按钮录入新的健康指标
- 系统自动检测异常指标并生成提醒

#### 随访记录
- 以卡片形式展示历史随访详情
- 包括主诉、检查结果、诊断、治疗方案、用药建议、健康指导、下次复诊日期
- 点击「新增记录」创建新的随访记录

#### 随访计划
- 查看该患者所有随访计划
- 查看计划状态（待随访、已完成、已取消）
- 点击「新增计划」创建新的随访安排

## 五、随访日程

点击左侧菜单「随访日程」进入日历视图：

1. **月份导航**：
   - 点击「‹」查看上一月
   - 点击「›」查看下一月
   - 点击「今天」快速回到当前月份

2. **日历视图**：
   - 每个日期格子显示当天的随访计划
   - 蓝色：待随访（PENDING）
   - 绿色：已完成（COMPLETED）
   - 灰色：已取消（CANCELLED）

3. **当日计划详情**：
   - 点击日期格子，右侧显示当天所有随访计划
   - 点击计划可编辑、标记完成或删除

4. **新增随访计划**：
   - 点击「新增计划」按钮
   - 选择患者、计划日期、随访频率（每天/每周/每两周/每月/每季度）
   - 填写备注信息

## 六、异常提醒

点击左侧菜单「异常提醒」查看所有异常指标：

1. **筛选功能**：
   - 按严重程度筛选：紧急(CRITICAL)、高(HIGH)、中(MEDIUM)、低(LOW)
   - 按状态筛选：未解决、已解决

2. **统计概览**：顶部显示各严重程度的未解决提醒数量

3. **提醒列表**：
   - 按严重程度从高到低排序
   - 红色背景：紧急
   - 橙色背景：高
   - 黄色背景：中
   - 蓝色背景：低

4. **操作**：
   - 「标记已读」：标记为已查看
   - 「标记已解决」：标记异常已处理
   - 「查看详情」：查看提醒完整信息
   - 「查看患者」：跳转至患者详情页

5. **批量操作**：
   - 勾选多条提醒
   - 批量标记已读或已解决

## 七、指标趋势图

点击左侧菜单「指标趋势图」查看健康数据可视化：

1. **筛选条件**：
   - 选择患者（必填）
   - 选择指标类型（血压、空腹血糖、餐后血糖、心率、胆固醇、甘油三酯、体重，可多选）
   - 选择时间范围（最近7天/30天/90天/180天/自定义）

2. **图表展示**：
   - 折线图展示指标变化趋势
   - 虚线表示正常范围上下限
   - 鼠标悬停可查看具体数值
   - 图例显示各指标颜色

## 九、常见问题

### Q1: 忘记密码怎么办？
请联系系统管理员重置密码。

### Q2: 如何批量导入健康记录？
目前系统支持单条录入，批量导入功能可在后续版本中扩展。

### Q3: 异常提醒的严重程度如何判定？
系统根据指标偏离正常范围的比例自动判定：
- 偏离 < 10%：低(LOW)
- 偏离 10%-30%：中(MEDIUM)
- 偏离 30%-50%：高(HIGH)
- 偏离 > 50%：紧急(CRITICAL)

### Q4: 数据多久备份一次？
建议每天自动备份一次，可参考部署说明配置定时备份任务。
